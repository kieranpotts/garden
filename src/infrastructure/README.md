# Infrastructure

This section covers the administration, configuration and upkeep of computer systems, including physical web servers and virtual ones, and other "cloud" services. These specialisms have historically been the responsibility of "IT operations" departments. Modern job roles include "infrastructure engineering" and "platform engineering".
