= Prompt engineering

Prompt engineering is the practice of designing and refining inputs to a *link:./large-language-model.adoc[large language model]* to elicit better, more accurate, or more useful outputs.

Because LLMs are sensitive to how questions and instructions are phrased, the structure and content of a prompt can significantly affect the quality of the response.
