= Blob storage (aka. object storage)

Also known as binary large object storage, blob storage is an alternative name for *link:./object-storage.adoc[object storage]*. The two terms are used interchangeably. Blob storage / object storage stores data as discrete objects in a flat namespace, each identified by a unique key and accompanied by rich metadata. It is designed for large volumes of unstructured data such as images, videos, audio files, backups, and documents.

Example public cloud services include Amazon S3, Azure Blob Storage, Google Cloud Storage, and Oracle Object Storage.
