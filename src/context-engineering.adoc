= Context engineering

Context engineering is the practice of designing and curating the full context window provided to a *link:./large-language-model.adoc[large language model]* — including system prompts, retrieved documents, conversation history, tool outputs, structured data, and other information. The objective is to improve the quality, relevance, and accuracy of model outputs.

*Prompt engineering* is a subset of context engineering. Prompt engineering is concerned with the wording and structure of the system prompt only. Context engineering is the broader discipline. It encompasses everything that goes into the context window, not just how individual instructions are phrased.

As models are increasingly deployed in agentic and multi-step workflows, context engineering becomes more important than prompt engineering alone.
