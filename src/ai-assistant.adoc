= AI assistant

An AI assistant is a general-purpose conversational interface to one or more *link:./large-language-model.adoc[large language models]*. Users interact via a chat UI — asking questions, drafting content, analysing documents, writing and debugging code — without needing to access the underlying model directly via an API.

Most AI assistants are front-ends to frontier models developed by the same company. Some support multiple models or allow the user to choose between them. Many are expanding beyond pure text chat to support file uploads, image generation, voice interaction, and web search.

See also *link:./ai-coding-assistant.adoc[AI coding assistants]* for assistants specialised in programming tasks, and *link:./ai-agent.adoc[AI agents]* for assistants with autonomous tool use and multi-step reasoning capabilities.

== General-purpose AI assistants

* https://chatgpt.com/[ChatGPT] — OpenAI's flagship assistant, built on the GPT model family. The most widely adopted AI assistant as of 2026. Available via web, mobile, and desktop apps, with free and paid tiers.

* https://claude.ai/[Claude] — Anthropic's assistant, built on the Claude model family. Known for its long context window, nuanced reasoning, and strong writing capabilities. Available via web and mobile.

* https://gemini.google.com/[Gemini] — Google's assistant, built on the Gemini model family. Integrated with Google Workspace (Docs, Gmail, Sheets).

* https://copilot.microsoft.com/[Microsoft Copilot] — Microsoft's assistant, powered by OpenAI models. Integrated into Windows, Microsoft 365, and the Edge browser.

* https://grok.x.ai/[Grok] — xAI's assistant, integrated into the X (formerly Twitter) platform.

* https://www.meta.ai/[Meta AI] — Meta's assistant, built on the Llama model family. Integrated into WhatsApp, Instagram, Facebook, and Messenger.

== AI search assistants

AI search assistants combine LLM reasoning with real-time web retrieval to answer questions with cited sources — an application of *retrieval-augmented generation (RAG)*, see *link:./large-language-model.adoc[large language model]*) for an explanation of this.

Traditional web search engines have also been extended with AI search capabilities — Microsoft Bing via Copilot, and Google Search via AI Overviews — though these are integrated into the broader search experience rather than being standalone assistants.

* https://www.perplexity.ai/[Perplexity] — AI-native search engine that answers questions using live web search with cited sources. Increasingly resembles a general-purpose assistant in its interface and capabilities.

* https://you.com/[You.com] — AI search engine with support for multiple models and personalisation features.
