= AI coding assistant

An AI coding assistant is a software tool that helps developers write, understand, test, and refactor code using large language models.

An AI code assistant is typically a _front-end_ to one or more *link:./large-language-model.adoc[LLM]* services. The front-end provides the user experience (for example, editor integration, chat, patch application, and tool execution), while the underlying model service provides the language and reasoning capabilities.

The front-ends may be implemented via terminal-based interfaces (TUIs), standalone GUI applications, or via integrations into existing IDEs and other applications and services.

Some coding assistant vendors develop and host their own fine-tuned models, rather than relying entirely on third-party providers. Anthropic is an example. Its Claude models are fine-tuned for coding tasks, and the company provides its own first-party front-ends, such as Claude Code, which offers both terminal-based and GUI interfaces (including via IDE integrations).

At the simplest end, assistants provide autocomplete and inline suggestions. More advanced assistants can explain unfamiliar code, generate tests, propose patches, and analyse a codebase and answer questions about it. Some also have *link:./ai-agent.adoc[agentic]* properties. They can plan multi-step tasks, use tools such as terminal commands and test runners, and iteratively revise changes based on results and feedback – all with a high degree of autonomy.

Examples of AI coding assistant front-ends include the following TUI tools:

* https://aider.chat/[Aider] — Open-source terminal coding assistant that supports multiple model providers.
* https://github.com/yetone/avante.nvim[avante.nvim] — Integrated various GenAI services into Neovim.
* https://claude.ai/code[Claude Code] — Anthropic's first-party terminal-based agentic coding assistant.
* https://opencode.ai/[OpenCode] — Open-source terminal-based agentic coding assistant supporting 75+ LLM providers.

And the following GUI tools:

* https://www.augmentcode.com/[Augment Code] — Premium code generation services, VS Code and JetBrains integrations.
* https://bolt.new/[Bolt] — Build and deploy full-stack web applications using only prompt engineering.
* https://github.com/cline/cline[Cline] — VS Code coding assistant with tool use and support for multiple LLM providers.
* https://codeium.com/[Codeium] — Coding assistance-as-a-service, extensions for many IDEs and text editors.
* https://sourcegraph.com/cody[Cody] — Coding assistant for IDE integration.
* https://www.continue.dev/[Continue] — Coding assistant for VS Code and JetBrains.
* https://cursor.com/[Cursor] — AI-native code editor, widely adopted for agentic coding workflows.
* https://github.com/features/copilot[GitHub Copilot] — Currently the most widely adopted coding assistant.
* https://supermaven.com/[Supermaven] — Code assistance for VS Code and JetBrains, and also Neovim. Acquired by Cursor in 2024 and likely to be discontinued as a standalone product.
* https://replit.com/ai[Replit Agent] — AI coding assistant for Replit.
* https://v0.dev/[V0] — Text-to-design for React and Next.js front-ends, from Vercel.

Tools like Cursor are blurring the boundaries between AI coding assistants and *link:./integrated-development-environments.adoc[Integrated development environments]*.
