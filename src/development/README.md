# Software development

This section covers tools and processes that support the development of _all_ software systems. In general, the topics and resources covered in this section are agnostic about **[computer languages](//github.com/kieranpotts/garden/tree/dev/src/languages)** and **[software platforms](//github.com/kieranpotts/garden/tree/dev/src/platforms)**. This section is concerned with the all phases of the software development life cycle and is not constrained to the discipline of computer programming, which is just one part of that cycle.
