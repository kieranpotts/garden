= Agent skills

Agent skills (also called tools or actions) are capabilities that allow AI agents to interact with external systems, perform tasks, and access data. They form the bridge between an agent's reasoning and real-world execution.

Agent skills can include:

* Data retrieval and queries
* API calls and integrations
* File and document operations
* System commands and automations
* Real-time information access

The *https://agentskills.io/[Agent Skills specification]* is a standardized format for defining and sharing agent skills, making them interoperable across different AI agent frameworks and platforms.
