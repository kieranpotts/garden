= AI agent

Agentic systems are systems that can act autonomously to achieve goals. They are a key component of many AI applications, from personal assistants to autonomous vehicles and software automation.

Common components of an AI agent include:

* *link:./large-language-model.adoc[Models]*
* *link:./agent-skills.adoc[Skills and tools]*
* Data, knowledge, and memory
* Input and output interfaces (for example, chat, audio, or APIs)
* Guardrails
* Orchestration

== Agent development frameworks

Frameworks and SDKs provide the infrastructure for building and orchestrating agents.

=== General-purpose frameworks

* https://github.com/langchain-ai/langchain[LangChain] — Comprehensive framework for building agents and LLM-powered applications. Emphasizes model interoperability, rapid prototyping, and an extensive ecosystem of integrations. Available in Python and TypeScript.

* https://github.com/run-llama/llama_index[LlamaIndex] — Framework specialized in data ingestion, organization, and retrieval-augmented generation (RAG). Bridges enterprise data and LLM capabilities with 300+ integration packages.

* https://github.com/microsoft/semantic-kernel[Microsoft Semantic Kernel] — Model-agnostic SDK for building and orchestrating AI agents and multi-agent systems. Emphasizes plugin architecture and enterprise features. Available in Python, .NET, and Java.

=== Vendor-specific SDKs

* https://developers.openai.com/api/docs/guides/agents-sdk[OpenAI Agents SDK] — Official SDK from OpenAI for building agents with their models.

* https://google.github.io/adk-docs/[Google Agent Development Kit (ADK)] — Official toolkit from Google for agent development.

* https://github.com/microsoft/agent-framework[Microsoft Agent Framework] — Production-grade framework for multi-agent workflows. Includes workflow orchestration, observability, and YAML-based agent definitions. Available in Python and C#/.NET.

=== Low-code and specialized platforms

* https://n8n.io/[n8n] — Low-code automation platform with https://docs.n8n.io/hosting/starter-kits/ai-starter-kit/[AI starter kit] for bootstrapping agent development environments.

* https://openclaw.ai/[OpenClaw] — Open-source, self-hosted agent platform with 15+ messaging channel integrations and support for 40+ LLM providers.

* https://pi.dev/[Pi] — Minimal agent foundation providing core components for building custom agents.

Many *link:./ai-coding-assistant.adoc[AI coding assistants]* have agentic properties, too.

== Agent orchestration platforms

Agent orchestration platforms sit above individual agent frameworks. Rather than providing tools to build a single agent, they coordinate teams of agents working together toward shared goals — managing scheduling, delegation, budgets, and governance across a whole system of autonomous workers.

* https://github.com/paperclipai/paperclip[Paperclip] — Open-source, self-hosted platform for running organisations of AI agents, with org charts, heartbeat-based scheduling, per-agent cost budgets, and full audit logging.

== Benchmarks

* https://www.tbench.ai/[Terminal Bench] benchmarks AI agent front-ends (user interfaces) for terminal environments, backed by various models.
