# Digital Garden

This is my digital garden – a sort of personal wiki or knowledgebase, a collection of bookmarks and notes and ideas that are in a perpetual state of work-in-progress.

## Contents

- **[Software development](./src/development/)** \
  Tools and processes to support the development of software systems. This section covers all phases of the software development life cycle and is not constrained to the discipline of computer programming.

- **[Software platforms](./src/platforms/)** \
  Runtime environments for software systems.

- **[Computer languages](./src/languages/)** \
  Programming, scripting, command, machine, assembly, query, and other computer languages.

- **[Data and encoding](./src/data/)** \
  Database management systems, data interchange protocols, and data engineering.

- **[Communication protocols](./src/protocols/)** \
  Systems for exchanging data between computing devices.

- **[Infrastructure](./src/infrastructure/)** \
  The administration, configuration and upkeep of computer systems, including web servers and cloud services.

- **[User interfaces](./src/ui/)** \
  User experience design (UXD) in digital products, plus wider coverage of human-computer interface design, both graphical and textual.

- **[Computer science](./src/cs/)** \
  Algorithms, data structures, and other fields of study in information technology.

- **[Work and business](./src/work/)** \
  Industry trends, business regulations, startups, and financing.

- **[Apps and services](./src/apps/)** \
  Application software, digital services and productivity tools.
